# Unicode Escape Sequence Decoder

Утилита для автоматического декодирования escape-последовательностей в текстовых файлах.

## Описание

**Unicode Escape Sequence Decoder** — это Python-скрипт, который автоматически обнаруживает и декодирует различные типы escape-последовательностей в текстовых файлах. Программа сканирует текущую директорию, находит все поддерживаемые текстовые файлы и создаёт их декодированные версии.

## Возможности

- **Декодирование Unicode escape-последовательностей** — преобразует `\uXXXX` в соответствующие символы Unicode
- **Декодирование Hex escape-последовательностей** — преобразует `\xXX` в символы по шестнадцатеричному коду
- **Обработка JSON слешей** — преобразует `\/` в обычный слеш `/`
- **Автоматическое определение кодировки** — поддерживает UTF-8, UTF-8-sig, cp1251, cp866, latin-1
- **Широкая поддержка форматов** — работает с 40+ расширениями текстовых файлов

## Установка

```bash
# Клонировать репозиторий
git clone https://github.com/baslie/unicode-decoder.git

# Перейти в директорию проекта
cd unicode-decoder
```

## Использование

### Windows

Запустите batch-файл:
```cmd
run_decoder.bat
```

### Linux / macOS

Запустите Python-скрипт напрямую:
```bash
python unicode_decoder.py
```

или

```bash
python3 unicode_decoder.py
```

## Результаты работы

Программа создаёт новые файлы с суффиксом `_decoded`:

**Пример:**
- `input.txt` → `input_decoded.txt`
- `data.json` → `data_decoded.json`
- `script.js` → `script_decoded.js`

Исходные файлы остаются без изменений.

## Пример работы

**Входной файл** (`example.txt`):
```
\u041f\u0440\u0438\u0432\u0435\u0442, \u043c\u0438\u0440!
Hello\x20World
Path: C:\/Users\/Roman
```

**Выходной файл** (`example_decoded.txt`):
```
Привет, мир!
Hello World
Path: C:/Users/Roman
```
